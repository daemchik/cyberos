# CyberOS — бэклог задач

## Смены и доступ

- [ ] Открытие / закрытие смены только по **реальным данным** API (касса, сотрудник, время, суммы).
- [ ] Пока смена **не открыта**: запретить операции клуба (карта, сессии, клиенты, маркет, задачи и т.д.).
- [ ] Пока смена **не открыта**: разрешить только отчёты, настройки ядра, конфиг тарифов, промокоды (уточняется по ТЗ).

## Тема, язык, типографика

- [ ] Переключение **тёмная / светлая** тема реально меняет оформление приложения.
- [ ] Переключение **языка** (RU/EN) работает по всему админ-интерфейсу.
- [ ] **Шрифт и размер** из настроек применяются глобально.

## Настройки и поведение админки

- [ ] **Звуковые уведомления** по ключевым событиям (заказ, вызов админа и т.д.).
- [ ] **Автовыход при бездействии** с сохранением последних действий (вкладка, черновики).
- [ ] **Пинг** до API — реальный RTT, не статический текст.
- [ ] **Индикатор нагрузки** — реальные доступные метрики (браузер/сервер по возможности).

## Конфиг и зоны

- [ ] В конфиге тарифов выбор **зоны из выпадающего списка** (данные с API).

## Игрок `/player`

- [ ] Вход: **телефон + пароль** обязательны.
- [ ] Исправить **вывод** на экране игрока (лишние символы, суммы, тестовые подписи, `?ws=`).

## Отчёты и аналитика

- [ ] **Графики** — читаемые оси, заливка/сетка, не «случайные линии».

## Задачи и клиенты

- [ ] Задачи админа: **приоритет** (создание, редактирование, фильтр).
- [ ] База клиентов: **редактирование, удаление**, управление балансом через UI.

## Уведомления

- [ ] Вкладка **«Уведомления»**: вызов администратора, заказы еды и др. события из API.

## Интеграции (отложено)

- [ ] WoL / перезагрузка / RDP / скрин на карте — подключить к API вместо заглушек.
